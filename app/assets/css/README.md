# 背景样式使用说明

## 当前使用

项目当前使用的是 **第 6 种背景样式：极简黑白灰** (`bg-beauty-minimal`) + **Plan B 装饰元素**

### Plan B: 极简装饰系统 ✨

在保持纯净白色背景的同时，通过以下微妙装饰元素平衡"干净"与"有趣"：

#### 1. 顶部装饰线
- **样式**: 1px 渐变线，使用项目主色（森林绿 `hsl(142 32% 32%)`）
- **位置**: 页面顶部固定
- **效果**: 从透明 → 主色 → 透明的水平渐变，极其微妙

#### 2. 页面边缘光晕
- **样式**: 左右两侧径向渐变
- **颜色**: 使用主色，3% 透明度（几乎看不见）
- **效果**: 在页面边缘营造轻微光影，增加层次感

#### 3. 圆形进度环按钮 ⭐ 当前启用
- **位置**: 更靠近内容区域（移动端：右下 `right-4 bottom-20`，桌面端：`right-8 bottom-8`）
- **样式**: 
  - SVG 圆环直径：56px（更大更明显）
  - 进度环半径：26px
  - 按钮尺寸：44px
  - 图标尺寸：20px（更小更精致）
  - 进度环线宽：2px（更细）
- **效果**: 环形进度 0-100% 填充
- **优势**: 
  - 节省空间，视觉新颖
  - 更靠近内容，易于点击
  - 进度环更宽，更容易看清
  - 整体尺寸紧凑

> 💡 侧边进度条方案已禁用（不够显眼）

#### 4. 卡片交互增强
- **Hover 效果**:
  - 边框颜色从灰色变为主色（30% 透明度）
  - 卡片轻微上浮（-0.5px）
  - 出现微妙阴影
- **过渡动画**: 300ms 平滑过渡

**设计哲学**：
- ✅ 99% 区域保持纯净白色，不影响阅读
- ✅ 1% 的装饰提供视觉焦点和趣味性
- ✅ 使用项目品牌色（森林绿），强化视觉识别
- ✅ 静态时极简，交互时活泼
- ✅ 所有装饰都是微妙的，不干扰内容
- ✅ 进度环更靠近内容，更易操作

**实现文件**：
- `app/assets/css/backgrounds.css` — CSS 装饰样式定义
- `app/layouts/default.vue` — 首页布局应用
- `app/layouts/post-layout.vue` — 文章页布局应用
- `app/pages/index.vue` — 首页卡片交互增强

## 可用的背景样式

在 `backgrounds.css` 中提供了 6 种美观的纯 CSS 背景样式：

| 样式名称 | CSS 类名 | 颜色特征 | 动画效果 | 视觉风格 |
|---------|---------|---------|---------|---------|
| 1. 柔和渐变网格 | `bg-beauty-gradient-mesh` | 青、粉、黄多层渐变 | 20秒缓慢移动 | 最柔和现代，色彩丰富 |
| 2. 现代极光 | `bg-beauty-aurora` | 紫、粉、蓝、绿四色 | 20秒色相旋转 | 大面积光晕，类似 macOS 壁纸 |
| 3. 优雅波纹 | `bg-beauty-wave` | 青、粉色线网格 | 30秒流动 | 极简专业，细腻纹理 |
| 4. 散点星空 | `bg-beauty-stars` | 七彩散点（蓝粉绿橙紫黄） | 20秒闪烁 | 多彩圆点分布，梦幻浪漫 |
| 5. 清新气泡 | `bg-beauty-bubbles` | 青橙紫红绿五色气泡 | 25秒浮动 | 大型圆形渐变，活泼清新 |
| 6. 极简黑白灰 | `bg-beauty-minimal` | 纯黑白灰 | 无动画 | 高级质感，最适合阅读 ⭐ 当前使用 |

### 各背景风格详解

**1. 柔和渐变网格** - 最适合需要色彩丰富但不失优雅的场景
- 三层渐变叠加（青、粉、黄）
- 背景位置动态移动，营造流动感
- 色彩过渡自然，不会抢夺内容焦点

**2. 现代极光** - 大面积柔和光晕，给人宁静感
- 四个大型径向渐变从不同方向发散
- 色相旋转动画让颜色持续变化
- 适合希望背景"有存在感"但不花哨的场景

**3. 优雅波纹** - 最低调，适合专业/极简风格
- 重复的细线网格，透明度极低
- 网格缓慢流动，细节精致
- 几乎不影响内容阅读，背景质感明显

**4. 散点星空** - 色彩最多，适合活泼/创意场景
- 七个不同颜色的圆形渐变点
- 整体透明度变化（闪烁效果）
- 像星空一样的散点分布

**5. 清新气泡** - 最"轻盈"的感觉，适合清新/年轻风格
- 五个大型气泡渐变
- 气泡位置浮动，像水中气泡
- 颜色饱和度适中，清新不刺眼

**6. 极简黑白灰 + Plan B 装饰** - 最高级、最适合阅读的背景 🎯
- **零色彩干扰**：只使用黑、白、灰，绝对不影响阅读
- **品牌色点缀**：微妙的森林绿装饰，强化视觉识别
- **渐变层次**：从白到浅灰的自然过渡（亮色模式）
- **完美暗色模式**：暗色模式下从深黑到深灰，眼睛最舒适
- **交互动效**：卡片 hover 时的微妙反馈
- 适合：博客文章、长篇阅读、技术文档、专业内容

## 如何切换背景样式

在以下文件中修改根 div 的 class：

### 1. `app/layouts/default.vue` (第 29 行)
```vue
<!-- 当前 -->
<div class="min-h-screen bg-beauty-minimal flex flex-col font-sans text-zinc-900 dark:text-zinc-50">

<!-- 切换为其他样式，例如柔和渐变网格 -->
<div class="min-h-screen bg-beauty-gradient-mesh flex flex-col font-sans text-zinc-900 dark:text-zinc-50">
```

### 2. `app/layouts/post-layout.vue` (第 42 行)
```vue
<!-- 同样的方式切换 -->
<div class="min-h-screen bg-beauty-gradient-mesh flex flex-col font-sans text-zinc-900 dark:text-zinc-50">
```

**注意**：如果切换到其他背景样式，Plan B 装饰元素（顶部线、边缘光晕、进度条）仍会显示。如需移除装饰，需要从布局文件中删除对应的 div 元素。

## 技术特点

- ✅ **纯 CSS 实现**：无需任何图片或外部资源
- ✅ **零性能影响**：使用 CSS 渐变和动画，GPU 加速
- ✅ **暗色模式适配**：所有样式都完美支持 dark mode
- ✅ **响应式设计**：在所有屏幕尺寸下表现良好
- ✅ **平滑动画**：subtle 动画不影响阅读体验
- ✅ **轻量级**：总文件大小 < 10KB

## 其他布局文件

如果你想在其他布局中也使用这些背景，可以修改：
- `app/layouts/group-post-layout.vue`
- `app/layouts/memo.vue`
- `app/layouts/admin.vue`
- `app/layouts/book.vue`
- `app/layouts/clean.vue`

只需将根 div 的 `bg-zinc-50` 替换为任意 `bg-beauty-*` 类名，并复制 Plan B 装饰元素即可。

### 🎯 推荐使用场景

- **博客首页/列表页** → 极简黑白灰 + Plan B（当前配置）⭐⭐⭐
- **文章详情页/长文阅读** → 极简黑白灰 + Plan B（当前配置）⭐⭐⭐
- **技术文档/教程** → 极简黑白灰 或 优雅波纹（专业感强）
- **创意/作品展示页** → 散点星空（活泼梦幻）
- **关于我/个人介绍页** → 现代极光 或 清新气泡（轻松亲切）
