# 插图计划 - Playful 风格

**文章**: 没用的旧代码不要丢，放到SKILL里继续用
**风格**: playful (有趣、创意、俏皮)
**插图数量**: 5 张

**风格说明**:
- 颜色：淡彩虹色、明亮跳跃、浅色背景
- 元素：涂鸦、有趣角色、对话气泡、表情符号风格图标
- 适合原因：文章语气轻松（emoji、开玩笑的口吻），playful 风格能让技术概念更有趣易懂

---

## 插图 1: 旧代码的新生

**插入位置**: "前言" 章节之后（第12行之后）
**目的**: 呼应"不要丢旧代码"的主题，用趣味方式展示代码复用
**视觉内容**:
- 左侧：一堆看起来废弃的代码片段（灰色、有灰尘）
- 中间：魔法棒或转换箭头
- 右侧：变成闪亮的 SKILL 盒子
- 可爱的表情：代码从😢变成😄
- 星星、闪光特效
**文件名**: illustration-code-rebirth.png

## 插图 2: SKILL 百宝箱

**插入位置**: "SKILL" 章节开始处（第14行之后）
**目的**: 展示 SKILL 的多样性和灵活性
**视觉内容**:
- 一个打开的宝箱
- 里面飘出三样东西：
  - 对话气泡（Prompt）
  - 卷轴（详细要求）
  - 工具箱（脚本）
- 可爱的火箭或机器人角色在旁边
- 彩色星星和小装饰
**文件名**: illustration-skill-treasure.png
