# 插图计划 - Notion 风格

**文章**: 没用的旧代码不要丢，放到SKILL里继续用
**风格**: notion (简约手绘线条、知识分享)
**插图数量**: 5 张

**风格说明**:
- 颜色：黑色轮廓、白色背景、1-2 个淡彩色点缀
- 元素：简单线条涂鸦、几何形状、手绘晃动感、大量留白
- 适合原因：文章是知识分享类内容，notion 风格简洁清晰，适合概念解释和思考过程展示
