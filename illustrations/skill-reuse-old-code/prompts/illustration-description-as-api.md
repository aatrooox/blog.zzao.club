插图主题：接口文档类比

风格：notion

视觉构成：
- 主视觉：左右对比，展示传统 API 和 SKILL Description 的类比关系
- 布局：
  - 左侧（传统 API）：
    - 文档图标（几页纸叠加）
    - 下方手写 URL："/api/xxx"
  - 右侧（SKILL）：
    - 文本框，内含 "Description" 字样
    - 旁边有灯泡图标（代表 Agent 理解）
  - 中间：手绘的等号或箭头连接，表示类比关系
- 装饰元素：灯泡图标用淡粉色填充，表示"理解"的瞬间；在文档图标上添加简单的横线表示文字

色彩方案：
- 主色：黑色（#1A1A1A）用于轮廓和文字
- 背景：纯白色（#FFFFFF）
- 点缀：淡粉色（#FADBD8）用于灯泡图标

文字内容：
- 左侧："/api/xxx"、"接口文档"
- 右侧：在文本框内 "Description"
- 灯泡旁："Agent 理解"
- 顶部可选标题："从 API 到 SKILL"

风格注释：用最简单的图标传达类比关系，强调 Description 如同接口文档的作用，手绘灯泡要有"啊哈时刻"的感觉，整体保持 Notion 风格的简洁和留白
