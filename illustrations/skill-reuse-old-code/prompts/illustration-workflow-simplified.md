插图主题：工作流简化

风格：notion

视觉构成：
- 主视觉：上下对比，展示传统多人协作 vs AI+SKILL 的效率差异
- 布局：
  - 上半部分（传统模式）：
    - 4 个简笔画小人图标，标记为 A、B、C、D
    - 用曲折的箭头线连接：A → B → B → C → D
    - 整体给人复杂、费时的感觉
  - 下半部分（AI 模式）：
    - 左侧：1 个简笔画小人
    - 中间：机器人图标或 "AI" 文字框
    - 右侧：3-4 个小方块并行排列（代表 SKILL 模块）
    - 用短直线连接，显示并行处理
  - 中间用向下的粗箭头或虚线分隔两部分
- 装饰元素：在 AI 图标处添加淡黄色点缀；传统模式的连接线要弯曲，AI 模式的线要笔直简洁

色彩方案：
- 主色：黑色（#1A1A1A）和深灰色（#4A4A4A）
- 背景：纯白色（#FFFFFF）
- 点缀：淡黄色（#F9E79F）用于 AI 图标

文字内容：
- 上半部分标注："A"、"B"、"C"、"D"
- 下半部分：小人旁可标注 "你"，AI 图标标注 "AI"
- SKILL 方块可标注："SKILL 1"、"SKILL 2"、"SKILL 3"
- 可选对比标签：上方 "传统"，下方 "AI 时代"

风格注释：通过对比强调效率提升，传统模式要显得复杂费时，AI 模式要显得简洁高效，使用 Notion 风格的极简手绘小人（火柴人风格），整体布局清晰明了
