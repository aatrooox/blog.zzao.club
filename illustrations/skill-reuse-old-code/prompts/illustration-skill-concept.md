插图主题：SKILL 概念图

风格：notion

视觉构成：
- 主视觉：画面中心是一个简单的方形框，内部手写风格标注 "SKILL"
- 布局：三条手绘线条从中心方框分别延伸到三个不同的图标
  - 左上：对话气泡图标（代表"意图描述"）
  - 右上：文档/纸张图标（代表"详细要求"）
  - 下方：代码符号 { }（代表"工具脚本"）
- 装饰元素：所有线条都带有手绘的轻微晃动感，营造手绘涂鸦风格；在文档图标处添加淡蓝色填充作为点缀

色彩方案：
- 主色：黑色（#1A1A1A）用于所有轮廓和线条
- 背景：纯白色（#FFFFFF）
- 点缀：淡蓝色（#A8D4F0）用于文档图标的填充

文字内容：
- 中心方框内："SKILL"
- 三个图标旁边分别标注：
  - "意图描述"
  - "详细要求"  
  - "工具脚本"

风格注释：使用 Notion 风格的简洁手绘线条，保持大量留白，所有元素都应该有手绘的不完美感（轻微晃动的线条），避免过于规整的几何图形
