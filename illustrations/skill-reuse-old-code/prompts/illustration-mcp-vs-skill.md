插图主题：MCP vs SKILL 流程

风格：notion

视觉构成：
- 主视觉：左右对比布局，展示两种不同的工作流程
- 布局：
  - 左侧（MCP）：
    - 票券/卡片图标 → 箭头 → 文字气泡（Prompt）→ 箭头 → 齿轮（调用）→ 箭头 → 结果框
    - 流程共 4-5 个步骤
  - 右侧（SKILL）：
    - 打包的盒子图标 → 粗箭头 → 结果框
    - 流程只有 2 个步骤
  - 中间：大大的手写 "vs" 字样分隔
- 装饰元素：MCP 侧的流程线较长且曲折，SKILL 侧的线条简短直接；在 SKILL 盒子上添加淡粉色填充

色彩方案：
- 主色：黑色（#1A1A1A）用于所有轮廓
- 背景：纯白色（#FFFFFF）
- 点缀：淡粉色（#FADBD8）用于 SKILL 盒子

文字内容：
- 左侧顶部："MCP"
- 右侧顶部："SKILL"
- 中间："vs"
- 流程中的简单标注："Prompt"、"调用"、"结果"

风格注释：通过流程长度的对比来强调 SKILL 的简洁优势，使用最少的元素，手绘风格的箭头要自然随意，避免过于工整
