插图主题：SKILL 像 Composable

风格：notion

视觉构成：
- 主视觉：三到四个方形模块图标，展示可组合的特性
- 布局：
  - 左侧：单独的小方块，标注 "Prompt"
  - 中间：单独的小方块，标注 "Script"
  - 右侧：两个方块组合在一起，标注 "Multi-Scripts"
  - 用手绘箭头或虚线显示它们之间的可组合关系
- 装饰元素：方块之间有简单的拼图式连接线，表示模块化；在 "Multi-Scripts" 处添加淡黄色填充

色彩方案：
- 主色：黑色（#1A1A1A）和深灰色（#4A4A4A）用于轮廓
- 背景：纯白色（#FFFFFF）
- 点缀：淡黄色（#F9E79F）用于 Multi-Scripts 模块

文字内容：
- "Prompt"
- "Script"
- "Multi-Scripts"
- 可选：顶部标题 "灵活组合"

风格注释：强调模块化和可组合性，使用最简单的方块和线条，手绘风格的连接线要有自然的不规则感，整体布局要有呼吸感（大量留白）
