DeepSeek 成了美国企业的救命稻草

Lindy 是一家旧金山的 AI 公司，做自动化工作流。

上个月他们的 AI 账单失控了。

主要用的是 Anthropic 的 Claude 模型，按 token 计费。随着业务量增长，每个月的 API 费用超过了员工工资。CEO Flo Crivello 说"这不可持续"。

本月初， Lindy 做了一个激进的决定： 100% 流量切换到 DeepSeek。

预计未来几个月能省下几百万美元。

这不是个例。越来越多的美国 AI 公司开始采用"模型路由"策略——不再把所有任务都扔给最贵的前沿模型，而是根据任务复杂度匹配不同模型。

简单的分类任务用便宜的小模型，复杂的推理任务才用 Claude 或 GPT-4。

DeepSeek 在这个趋势中成了赢家。它的 V3 模型性能接近 GPT-4 ，但价格只有十分之一。API 定价是 $0.27/百万输入 token ，$1.10/百万输出 token。

相比之下， Claude 3.5 Sonnet 是 $3/$15 ， GPT-4o 是 $5/$15。

差距太大了。

一些公司甚至开始暂停 AI 投入，等证明投资回报率后再继续。AI 的"先烧钱再说"时代，正在结束。

DeepSeek 的崛起还有一个微妙之处：它是中国公司。

美国企业大规模转向中国 AI 模型，这在两年前不可想象。但现在，成本压力压倒了地缘政治考量。

当然，数据安全仍是顾虑。有些公司要求 DeepSeek 的 API 必须走美国境内的服务器，或者只用开源版本本地部署。

但至少现在，当 CFO 问"为什么 AI 部门这么烧钱"时， CTO 有了一个答案：我们可以换 DeepSeek。
