昨天深夜，马斯克的 xAI 扔了一颗炸弹。

Grok Build——他们两个月前刚发布的编程 Agent——**完全开源了**。GitHub 仓库 `xai-org/grok-build` 全部公开，核心代码、终端界面、扩展系统，一个不少。而且，所有用户的使用额度**全部重置**。

这意味着什么？任何人，一分钱不花，就能在自己电脑上跑一个能和 Claude Code、Codex CLI 掰手腕的 AI 编程助手。

「白嫖」编程 Agent 的时代，真的来了。

## 隐私翻车逼出的开源

这件事的起因，说实话，不太光彩。

Grok Build 今年 5 月 14 日首发，当时只对付费用户开放。定位很明确：一个终端里的 AI 编程助手，能读代码、写代码、搜代码、跑代码，和 Claude Code 正面对标。

但 6 月份，安全研究人员发现了一个大问题——**Grok Build 会在用户不知情的情况下，把整个代码仓库同步到 Google Cloud 服务器**。包括私有仓库、内部项目、还没发布的商业代码。

这事一出，开发者圈子炸了。

做企业软件的团队，涉及合规要求的项目，谁敢用？ xAI 倒是反应快：删除了所有已上传的用户数据，关闭了云端数据留存。但信任这东西，破了就是破了。

**开源，成了修复信任的唯一出路。**

## 开源包里到底有什么？

这不是那种阉割版的「开源」。xAI 这次放出来的东西相当实在：

- **完整的 Agent 核心循环**：怎么规划任务、怎么调度工具、怎么执行操作，全部可见
- **所有代码交互工具**：读文件、编辑代码、搜索、执行，一个不少
- **终端 UI 界面**： TUI 交互层完整开源
- **扩展系统**： skills、plugins、hooks、MCP 服务器、子 Agent ，全部公开

简单说，你拿到的就是一个**功能完整的编程 Agent**。唯一需要的是你自己的 API key——模型推理还是要走 xAI 的 API ，但代码**绝不会离开你的机器**。

这和之前云端的版本有本质区别。本地跑意味着三件事：**隐私可控、零使用上限、完全可定制**。

## 编程 Agent 三国杀

Grok Build 开源，把 2026 年编程 Agent 的竞争推到了新高度。

现在的局面是这样的：**Anthropic 的 Claude Code** 靠着先发优势和生态积累，据说已经帮 Anthropic 做到了 140 亿美元的年经常性收入，是这个赛道里最赚钱的产品。**OpenAI 的 Codex CLI** 上线一个月就破了百万开发者，增速惊人。

Grok Build 是老三，但打法最野。

它的看家本领是**Arena Mode （竞技场模式）**——同时启动最多 8 个并行 Agent ，每个独立完成任务，然后自动评分排名，把最好的方案推到开发者面前。你不用一个个对比，系统替你挑了最优解。

底层模型 **grok-code-fast-1** 是从零训练的编程专用模型， SWE-Bench 得分 70.8%，输入价格只要**每百万 token 0.2 美元**。这个定价，比 Claude 和 GPT 的编程接口便宜一大截。

## 白嫖时代的真正含义

Grok Build 开源这件事，真正的冲击不在技术层面。

**它改变了编程 Agent 的分发逻辑。**

之前， Claude Code 和 Codex CLI 虽然也各有免费入口，但核心体验都绑着付费墙。你要么按月订阅，要么按 token 烧钱。Grok Build 开源后，**本地运行零成本，你只需要付 API 调用的钱**——而且 grok-code-fast-1 本身就很便宜。

更关键的是，开源意味着**任何人都能 fork、魔改、分发**。可以预见，很快会出现各种定制版的 Grok Build ：针对前端的、针对数据分析的、针对 DevOps 的。生态一旦起来，闭源产品很难招架。

AI 编程这件事，正在从「买软件」变成「搭积木」——开源 Agent 做框架，便宜的 API 做引擎，开发者自己组装。

以前你要花 20 美元/月订阅一个编程助手。以后你可能只需要几美分的 API 费，就能跑一整天。

## 额度重置的小心思

xAI 还同时宣布**重置了所有用户的使用额度**。

官方的解释是「发现了缓存效率问题，导致用户过早触达上限」。但明眼人都看得出来，这是一步聪明的棋——**开源吸引新用户，额度重置留住老用户**。先让你免费爽一把，爽完了自然会想掏钱。

毕竟 xAI 知道，开源只是开始。真正的战场在模型能力。Grok 4.5 在 7 月 8 号刚刚发布，已经是 Grok Build 的默认模型。**开源抢地盘，模型赚利润**，这个打法很马斯克。

## 开发者该怎么选？

如果你在乎隐私、预算有限、或者想深度定制自己的编程助手——**Grok Build 现在是最优解**。本地跑、免费改、便宜用。

如果你依赖成熟的 IDE 集成和丰富的第三方插件——**Claude Code 仍然是体验最好的选择**。毕竟人家的生态积累了两年多。

如果你是 OpenAI 全家桶用户——**Codex CLI 的整合体验最丝滑**。

但不管你站哪边，有一个趋势已经不可逆：**AI 编程工具正在从奢侈品变成日用品**。

而马斯克，亲手给这个过程踩了一脚油门。
